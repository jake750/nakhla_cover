# 🌴 Nakhla Cover: Sustainable Shading Solutions

**Nakhla Cover** is a premium, eco-friendly architectural project rooted in the heritage of **Gabes, Tunisia**. This repository hosts the official landing page for our sustainable shading panels, designed to provide natural passive cooling while maintaining a high-end "Boho-Chic" aesthetic.

## 🚀 Live Demo
Check out the live site here: 
**[https://jake750.github.io/nakhla_cover/](https://jake750.github.io/nakhla_cover/)**

---

## ✨ Features
*   **Eco-Friendly Design**: 100% natural palm fronds sourced from Tunisian oases.
*   **Passive Cooling**: Natural thermal insulation that reduces energy consumption.
*   **Premium Aesthetic**: Integrated warm LED ambiance for a modern look.
*   **SDG Aligned**: Directly supports **SDG 13 (Climate Action)** and **SDG 14 (Life Below Water)** by providing a plastic-free alternative.
*   **Dark Mode Optimized**: A sleek, modern UI designed for high visual impact.

## 🛠️ Technical Stack
*   **Frontend**: HTML5, CSS3, JavaScript.
*   **Frameworks**: Responsive design for mobile and desktop.
*   **Hosting**: GitHub Pages.
